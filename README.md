# HTML Learning Journey

A collection of HTML practice files and projects built from scratch while learning web development. This repo documents my progress from basic HTML structure to building multi-page websites — all using only HTML (no CSS or JavaScript).

## Projects

| Project | Description |
|---------|-------------|
| [Personal Portfolio](projects/portfolio.html) | Portfolio page with about, skills, and contact sections |
| [Chicken Biryani Recipe](projects/recipe.html) | Recipe page using lists and semantic structure |
| [Learning Blog](projects/blog.html) | Blog layout with multiple articles using semantic tags |
| [Class Timetable](projects/timetable.html) | Weekly schedule table with colspan for merged periods |
| [Contact Form](projects/contact.html) | Form with input fields, select dropdown, textarea, and fieldset |
| [Apple Landing Page](projects/apple-landing.html) | Mock product landing page |
| [Signup Form](projects/signup.html) | Email and password signup form |
| [Multi-page Website](projects/website/) | 4-page website (Home, About, Services, Contact) with shared navigation |

## Practice Files

Hands-on exercises covering essential HTML concepts:

- HTML structure and doctype
- Headings, paragraphs, text formatting
- Links and images
- Ordered and unordered lists
- Tables with colspan and rowspan
- Forms with various input types
- Semantic HTML (header, nav, main, article, section, footer)
- Div and span containers
- Form elements (textarea, select, fieldset, radio, checkbox)

## What I Learned

- Write clean, well-structured HTML documents
- Use semantic tags for meaningful content organization
- Build forms with proper labels and input types
- Create multi-page websites with navigation
- Use tables for structured data display
- Apply version control with Git and meaningful commit messages
